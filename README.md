# Telegram-бот «Школа онлайн-профессий»

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

1. Создай бота через [@BotFather](https://t.me/BotFather) и получи токен.
2. В файле `bot.py` замени строку:
   ```python
   TOKEN = "YOUR_BOT_TOKEN_HERE"
   ```
   на свой токен, например:
   ```python
   TOKEN = "7123456789:AAF..."
   ```
3. Запусти бота:
   ```bash
   python bot.py
   ```
